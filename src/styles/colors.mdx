# Colors

主要な色です。

色はSassの変数([source](https://github.com/Japan-Venture-Research/plume-ui/blob/master/src/styles/_colors.scss))とJavaScriptの変数([source](https://github.com/Japan-Venture-Research/plume-ui/blob/master/src/styles/colors.ts))としてそれぞれexportされています。

<React.Fragment>
  {Object.entries(colors).map(([name, code], i) => {
    // Skip exceptional `__filemeta` appended by Docz
    if (name === '__filemeta') {
      return
    }
    return <ColorSwatch name={name} code={code} key={i} />
  })}
</React.Fragment>
