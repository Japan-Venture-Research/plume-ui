# Font

font-familyに指定するフォントをSassの変数([source](https://github.com/Japan-Venture-Research/plume-ui/blob/master/src/styles/_font.scss))とJavaScriptの変数([source](https://github.com/Japan-Venture-Research/plume-ui/blob/master/src/styles/font.ts))として定義しています。

global-fontがベースのフォント、display-fontが英文の見出しに利用する特徴的なフォントです。
