# Divider

コンテンツ間に線を引く役割です。

## CSS

`pl-divider`クラスを使います

<Playground>
  <hr className="pl-divider" />
</Playground>

### Dashed

点線のスタイルにします。

<Playground>
  <hr className="pl-divider is-dashed" />
</Playground>

  <Divider />
</Playground>

### Dashed

点線のスタイルにします。

  <Divider dashed />
</Playground>
