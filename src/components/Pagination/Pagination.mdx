# Pagination

複数のページ間を遷移するためのコンポーネントです

## CSS

- ページネーション全体を`pl-pagination`で囲みます。
  - `pl-pagination-previous` `pl-pagination-next`で前・次のページへの遷移導線を描画します
    - 有効でない（ex. すでに最初・最後のページにいる）場合は`.is-disabled`クラスを付与します
  - `pl-pagination-list`でページネーションの要素を囲みます
    - `pl-pagination-item`でページネーションの要素を描画します。`is-active`で現在アクティブなページを表現します
    - `pl-pagination-ellipsis`で要素間の`...`を表現します

---

`pl-pagination-item`の直下に`a` / `div` / `span`などの要素を配置してください

    </a>

</Playground>

## React

- ページネーション全体を`<Pagination />`で囲みます。
  - `<Pagination.Previous />` `<Pagination.Next />`で前・次のページへの遷移導線を描画します
    - 有効でない（ex. すでに最初・最後のページにいる）場合は`disabled` propsを付与します
  - `<Pagination.List />`でページネーションの要素を囲みます
    - `<Pagination.Item />`でページネーションの要素を描画します。`active`propsで現在アクティブなページを表現します
    - `<Pagination.Ellipsis />`で要素間の`...`を表現します

---

`Pagination.Item`の直下に`a` / `div` / `span`などの要素を配置してください

<Playground>
  <Pagination>
    <Pagination.Previous disabled href='#' />
    <Pagination.List>
      <Pagination.Item>
        <a href='#'>
          1
        </a>
      </Pagination.Item>
      <Pagination.Item>
        <a href='#'>
          2
        </a>
      </Pagination.Item>
      <Pagination.Ellipsis />
      <Pagination.Item active>
        <a href='#'>
          312
        </a>
      </Pagination.Item>
      <Pagination.Item>
        <a href='#'>
          313
        </a>
      </Pagination.Item>
    </Pagination.List>
    <Pagination.Next href='#' />
  </Pagination>
</Playground>
