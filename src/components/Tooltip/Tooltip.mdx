# Tooltip

ツールチップは、補足的な情報を表示するためのコンポーネントです。ユーザーがマウスオーバーしたり、クリックした際に表示されます。

️❗️Tooltipはユーザーの能動的なアクションがないと表示されないため、単なる補足ではなくユーザーにとって重要性が高い情報に関してはツールチップ以外のUIでの表現を考慮すべきです️

## CSS

`pl-tooltip`クラスを使います。

<Playground>
    <Centered>
        <span className="pl-tooltip">
            こんばんは
            <span className="box top">こんにちは！</span>
        </span>
    </Centered>
</Playground>

## React

`Tooltip`コンポーネントでツールチップを表示するコンポーネントを囲みます。
`label`propsに表示するテキストを渡します。
`place`propsに表示する位置を渡します。
`long`propsは長い文章を表示する場合にtrueを渡します。

<Playground>
    <Tooltip label="なんと！たからばこはミミックだった！" place="top">
       たからばこ
    </Tooltip>
</Playground>
