# NewTab

どのコンテンツが現在表示されているかを表現し、コンテンツを切り替えるためのコンポーネントです。

## CSS

`pl-tab-group`で`pl-tab`のリストを囲みます。
`is-active`クラスで現在表示されているコンテンツを示します。

タブに数値を表示する場合は`pl-tab-count`クラスを使います。

<Playground>
  <div className="pl-tab-group">
    <a className="pl-new-tab is-active" href="#">
      企業概要
    </a>
    <a className="pl-new-tab is-disabled" href="#">
      ストーリー
    </a>
    <a className="pl-new-tab" href="#">
      投資先
      <span className="pl-tab-count">87</span>
    </a>
    <a className="pl-new-tab" href="#">
      スタートアップ投資
      <span className="pl-tab-count">1,023</span>
    </a>
  </div>
</Playground>

## React

`Tab.Group`で`Tab`コンポーネントのリストを囲みます。
`active`props で現在表示されているコンテンツを指定します。
タブに数値を表示する場合は`count`props に指定します。

<Playground>
  <NewTab.Group>
    <NewTab as="a" href="#">
      企業概要
    </NewTab>
    <NewTab as="a" href="#" disabled={true}>
      ストーリー
    </NewTab>
    <NewTab as="a" href="#" count={0}>
      投資先
    </NewTab>
    <NewTab as="a" href="#" count={1023} active>
      スタートアップ投資
    </NewTab>
    <NewTab as="a" href="#" count={0}>
      LP投資
    </NewTab>
  </NewTab.Group>
</Playground>
